# Unlimited-OCR 使用说明

## 位置

所有文件在：**D:\Python\OCR\**

| 文件 | 用途 |
|------|------|
| `ocr_web.py` | Web 界面（推荐） |
| `batch_ocr.py` | 批量脚本（高级） |
| `models/` | 模型文件（6.3GB） |

## 方式一：Web 界面（最简单）

```
1. 双击运行 启动Web界面.bat
2. 浏览器打开 http://localhost:5000
3. 拖拽 PDF → 设页码 → 点开始
```

## 方式二：命令行批量

修改 `batch_ocr.py` 第 12-14 行的路径，然后：
```
D:\Python\.venv\Scripts\python.exe D:\Python\OCR\batch_ocr.py
```

## 环境

- Python: `D:\Python\.venv\Scripts\python.exe`
- PyTorch 2.11 + CUDA 12.8
- 需要 NVIDIA 显卡（RTX 5060 或其他 8GB+ 显存）
- 不需要联网，全离线运行
