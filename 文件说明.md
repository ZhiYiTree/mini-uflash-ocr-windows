# 文件说明

根目录只保留当前运行和维护需要的入口。普通使用时，双击 `启动前端.bat` 即可。

## 日常使用

| 文件或目录 | 用途 | 是否需要保留 |
| --- | --- | --- |
| `启动前端.bat` | 普通用户双击启动网页 | 是 |
| `frontend.py` | 简洁的 Python 前端入口 | 是 |
| `launch_webapp.ps1` | 启动服务、打开浏览器、记录 PID | 是 |
| `stop_webapp.ps1` | 只停止本项目的网页进程 | 是 |
| `webapp/` | 当前 Gradio 网页和 OCR 业务代码 | 是 |
| `models/` | Unlimited-OCR 本地模型 | 是，不要删除 |
| `weights/` | Mini UFlash Stage 11B 权重 | 是，不要删除 |
| `.venv/` | 项目独立 Python 环境 | 是；可通过安装脚本重建 |

## 安装与维护

| 文件或目录 | 用途 |
| --- | --- |
| `install_windows.ps1` | 创建环境并安装 Windows 依赖 |
| `requirements-windows.txt` | Python 依赖版本 |
| `check_environment.ps1` | 检查 GPU、CUDA 和 Python 环境 |
| `download_models.ps1` | 检查或显式下载模型 |
| `scripts/` | 真实推理测试脚本 |
| `reports/` | 已验证的环境和推理报告 |
| `README.md` | 完整项目说明 |

## 自动生成，可随时清理

| 目录 | 内容 |
| --- | --- |
| `webapp/outputs/` | 每次识别的可下载结果 |
| `logs/` | 网页运行日志 |
| `__pycache__/`、`.pytest_cache/` | Python 和测试缓存 |
| `webapp.pid` | 服务运行时的进程编号 |

## 历史归档

`archive/` 不参与当前程序运行：

- `legacy-app/`：旧版 Flask 页面、批处理脚本及旧识别结果；
- `reference-sources/`：从备份中解出的研究源码，仅供查阅；
- `release-packages/`：原始压缩包、校验文件和发布清单；
- `project-notes/`：早期计划记录；
- `previous-runs/`：整理前保留下来的网页输出。

确认不再需要历史版本后，可以整体删除 `archive/`；这不会影响当前网页运行。
